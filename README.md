<h1>iRacing On Linux</h1>

<h2>GAME STATUS: KIND OF INOPERATIVE</h2>

Please do have a read of the **iracing-setup-check directory**, there is a script to support people looking to check if their system is set up correctly to run iRacing along with a manual walkthrough of the same process.  It can fully check the status of Steam, protontricks, iRacing's status within Steam as a Steam account or Direct Account added with a generated Steam key from iRacing, make sure all necessary libraries are present and helps with accessing iRacing files in Documents.

Test Drive, Replays, Time Attack and AI Racing can be accessed at your own risk.

A collaborative effort to record and document as much information about iRacing on Linux, peripherals, accompanying software, Linux distributions and more.

Each document regarding a certain distribution must be exclusive to that distribution, even if the upstream distribution is the same - for example clean Arch, Manjaro and EndeavourOS are all Arch however as they have different install processes they must not be combined.
